PRINTALITY: Grupo 9

Cambios realizados en la interfaz


  • Modificación del menú lateral. Se ha añadido el acceso a trabajos pendientes.
  • Modificación en las tablas de gestión de impresoras.
    • Nuevo botón para ver los grupos.
    • La celda de trabajos muestra únicamente la cantidad de trabajos.
  • Botón de añadir. Muestra una ventana emergente para introducir los campos necesarios.
  • Funcionalidad Vincular/Desvincular. Se gestiona únicamente desde la vista de gestión de impresoras.
  • El botón de imprimir muestra una ventana emergente para seleccionar los archivos. También puedes elegir la impresora o grupo que desees.
  • El botón Cancelar. Muestra una ventana emergente donde puedes elegir el trabajo que deseas detener.
  • Eliminado el botón de cancelar del menú superior.
  • Modificada la vista de impresión. Ahora utiliza input type file en vez de mostrar los trabajos disponibles.
  • Añadida funcionalidad para desvincular y vincular impresoras de grupos. Muestra una alerta y modifica la tabla.
  • Modal para editar grupo. Nueva ventana para modificar el nombre.
  • Modal para editar impresora. Nueva ventana para modificar alias, lugar e ip.
  • Sripts para eliminar filas en impresoras, grupos y trabajos pendientes.
  • Cuando se elimina una impresora
    • Se reduce el numero de impresoras del grupo al que pertenece.
    • Se eliminan los trabajos pendientes que tuviese.
  • Modificada la url del servidor y el nombre y contraseña de usuario.
  • Modificado placeholder de la barra de busqueda, varía en funcion de la tabla seleccionada(impresoras,grupos o trabajos).
  • Buscar filtra en todas las tablas.
  • Modificado el codigo de color para las tablas:
    • Verde y rojo para estados de impresión positivos y negativos(imprimiendo o sin papel).
    • Azul para los grupos a los que pertenece una impresora.
    • Negro para trabajos en cola.
    • Gris para las impresoras pertenecientes a un grupo(vista gestión de grupos).
  • Añadida funcionalidad al icono de editar(grupos e impresoras).
  • Scripts para borrar una fila de grupos, impresoras y trabajos.
  • El botón para imprimir trabajos solo acepta ficheros pdf.
  • Los botones de vincular y desvincular se activan y desactivan si:
    • Se intenta vincular una impresora a un grupo al que ya pertenece.
    • Se intenta desvincular una impresora a un grupo al que no pertenece.
